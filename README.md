# manufacturing-consent
Structured skepticism agent for analyzing framing, incentives, narrative drift, and overreach.
## Purpose

This project is designed to help distinguish:

* Verified Fact
* Interpretation
* Narrative Framing

It includes:

* Drift detection
* Incentive analysis
* Missing-context checks
* Anti-overreach safeguards

## Core Test Prompt

Lean check this headline:

Government’s New Safety Law Sparks Fears of Authoritarian Crackdown

Return:

* Signal Level
* Fact
* Interpretation
* Narrative
* Drift Check
* Bottom Line

Then critique your own answer for possible overreach.

## Failure Modes Being Watched

* Narrative inflation
* Pattern over-detection
* Incentive overreach
* False hidden-coordination inference

## Status

Early architecture / calibration stage.
